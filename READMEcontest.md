# Портфолио-бот | CapEducation Конкурс «ОБО МНЕ» 🏆

Привет! Меня зовут Артур, и это мой Telegram-бот, разработанный специально для участия в конкурсе от **CapEducation** (Трек: Middle Python). 

Бот представляет собой полноценное интерактивное портфолио, которое в удобной форме знакомит пользователя с моими навыками, проектами и целями в разработке.

## 🚀 Архитектурные фичи и технологии
Проект написан с упором на чистоту кода и использование продвинутых концептов:
* **Основной стек:** `pyTelegramBotAPI` (Telebot) для работы с Telegram API.
* **Гибкий запуск:** Реализована поддержка передачи токена через аргументы командной строки (`sys.argv`). При этом на GitHub сам токен скрыт в целях кибербезопасности.
* **Система логирования:** Настроено автоматическое логирование действий пользователей через модуль `logging`. Все логи пишутся в файл `logs.txt` с указанием времени, ID и имени пользователя.
* **Динамический контент:** Использование модуля `random` для выдачи случайных фотографий моих дипломов при каждом нажатии на кнопку «Достижения».
* **Интерфейс и UX:** Удобное Reply-меню на 9 блоков, поддержка Markdown-разметки для красивого оформления текста и пасхалка с анимированным кубиком (Dice) при отправке стикера.
* **Кастомизация терминала:** Интеграция библиотеки `colorama` для цветного вывода статуса запуска в консоль.
